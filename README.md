# Toy Supplier Sales & Inventory Analysis Dashboard

This project is a complete Power BI dashboard designed for analyzing sales performance, inventory value, store operations, and product insights for a toy supplier company.  
It demonstrates end-to-end BI development including data modeling, DAX calculations, and business reporting.

---

## Project Overview

This dashboard provides analysis of:

- Sales performance across all stores  
- YOY (Year-over-Year) sales and growth metrics  
- Monthly and yearly sales trends  
- Inventory retail value and inventory cost value  
- Gross margin performance  
- Fast-moving and dead stock identification  
- Ranking of top products, stores, and invoices  
- Store age (business age) and operational KPIs  
- Drill-down analysis using Decomposition Tree

---

## Key Features

1. **Sales Analytics**
   - Monthly and yearly trends
   - YOY Sales and YOY Growth measures
   - Top 5 invoices
   - Ranking of highest revenue stores

2. **Inventory Analytics**
   - Inventory retail value
   - Inventory cost value
   - Gross margin
   - Store-level inventory analysis

3. **Product Insights**
   - Fast-moving items
   - Dead stock items
   - Category-level sales contribution
   - Top revenue-generating products

4. **Store Operations**
   - Store age (Business Age)
   - Store count
   - Sales contribution by store and city

5. **Power BI Techniques Used**
   - Star schema data modeling  
   - DAX measures (YOY, YTD, MTD, ratios, margins)
   - Power Query data transformation
   - Conditional formatting and KPI design
   - Decomposition Tree for detailed analysis

## Data Model

The project uses a Star Schema structure:

- Fact table: **Sales**  
- Dimension tables: **Products**, **Stores**, **Inventory**  

This structure enables clean relationships and efficient DAX performance.
